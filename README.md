# 🎵 Spotify Web Player UI Clone
A Spotify-inspired music player interface built using HTML and CSS. This project recreates the layout of the Spotify Web Player, focusing on modern UI design, dark theme layout, and responsive component structure.

The goal of this project is to practice frontend layout building, UI structuring, and CSS styling techniques.

🚀 Features

✔ Spotify-like Dark Theme Interface

✔ Sidebar Navigation Menu

✔ Music Library Section

✔ Playlist / Album Cards Layout

✔ Music Player Control Bar

✔ Album Artwork Display

✔ Responsive Flexbox Layout

🧩 UI Sections

1️⃣ Sidebar Navigation

Contains key navigation items such as:

Home

Search

Your Library

This section mimics Spotify's left navigation panel.

2️⃣ Music Library

Displays:

Playlists

Recently played albums

Music cards with cover images

3️⃣ Content Area

The main section showing:

Featured playlists

Albums

Music recommendations

Each card contains:

Album artwork

Title

Artist description

4️⃣ Music Player Control Bar

The bottom music player includes:

Play / Pause controls

Previous / Next buttons

Progress bar

Album artwork display
